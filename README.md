# topology-demo

如何使用 es6+webpack 方式编译使用的 demo

# Development

## 1. 安装依赖库

```
[demo#] yarn
# 或
[demo#] npm install

```

## 2. 编写 es6

需要使用 topology 的业务逻辑在 index.js 中，通过下面命令编译成 es5 的文件，使 index.html 直接使用。

```
# build
[demo#] npm run build

```
